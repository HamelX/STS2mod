# GunslingerMod 카드 효과 정리 (KR)

> 기준: `GunslingerMod/GunslingerMod/localization/kor/cards.json`
> 
> 참고: 실제 수치(피해/방어/드로우)는 강화/동적 변수에 따라 게임 내에서 변동될 수 있습니다.

## 시작/기본 코어

- **사격 (SHOOT)**: 탄환 발사(업그레이드 시 2회). 빈 약실이면 회전만.
- **방어 (DEFEND_GUNSLINGER)**: 방어도 획득.
- **장전 (RELOAD)**: 일반탄 2발 장전(업그레이드 시 강화탄 2발).
- **회피 (EVASION)**: 이번 턴 받는 피해 절반(업그레이드 시 추가 1 감소).
- **난사 (SPRAY_FIRE)**: 장전된 탄환 전부 발사, 매 발 무작위 적 대상.
- **패닝 (PANNING)**: 장전된 탄환 전부를 대상에게 순차 발사.

## 봉인(Seal) 계열

- **봉인탄 장전 (SEAL_LOAD)**: 봉인탄 2발 장전, 가득 차 있으면 레벨 상승(업그레이드 시 봉인 보호 추가).
- **봉인 추적 (SEAL_SEARCH)**: 드로우 후 최고 레벨 봉인탄을 발사 칸으로 이동.
- **봉인 의식 (SEAL_RITE)**: 지속 파워. 턴 시작 시 장전된 봉인탄 레벨 상승 + 봉인탄 장전 시도.
- **봉인 개방 (SEAL_OPEN)**: 최고 레벨 봉인탄을 발사 칸으로 이동(업그레이드 시 레벨 +2).
- **봉인 증폭 (SEAL_AMPLIFY)**: 봉인탄 레벨 상승(단일 선택 또는 전체 상승).
- **봉인 공명 (SEAL_RESONANCE)**: 최고 봉인 레벨 기반 고정 배율 피해.
- **봉인 해방: 개 (SEAL_RELEASE_KAI)**: 모든 봉인탄 발사, 처치 시 타깃 재지정.
- **봉인 폭주 (SEAL_RAMPAGE)**: 선택 봉인탄 레벨 기반 에너지 획득 후 레벨 감소.
- **봉인 통찰 (SEAL_INSIGHT)**: 드로우 + 현재 칸 봉인탄/레벨 조건 추가 드로우.
- **그랜드 라이트 (GRAND_RITE)**: 장전된 봉인 레벨 전체 상승 후 현재 약실 1회 발사.

## 각인/도탄/표식 계열

- **처형 사격 (EXECUTION_SHOT)**: 1회 발사, 반피 이하 대상 피해 2배 + 각인 3.
- **도탄 사격 (RICOCHET_SHOT)**: 각인 소모 후 발사, 적 수에 따라 도탄 부여 분배.
- **현현: 도탄 (IMPRINT_MANIFEST_RICOCHET)**: 각인 소모 후 현현:도탄 부여.
- **낙인 흩뿌리기 (FAN_THE_BRAND)**: 도탄 부여 + 각인 획득.
- **표식 폭발 (TAG_BURST)**: 방아쇠 1회, 발사 시 추가 피해 + 도탄 부여.
- **표식 읽기 (READ_THE_MARK)**: 방어도 획득, 현현:도탄 적 존재 시 드로우+회전.
- **에코 노트 (ECHO_NOTE)**: 각인 소모 후 드로우.
- **각인 압착 (IMPRINT_SQUEEZE)**: 손패 1장 버리고 각인 획득(업그레이드 시 +1).
- **리바운드 넷 (REBOUND_NET)**: 모든 적 도탄 부여(단일 적일 때 강화).

## 트레이서/연사/운용 계열

- **추적탄 장전 (TRACER_LOAD)**: 추적탄 다수 장전.
- **워킹 파이어 (WALKING_FIRE)**: 방아쇠 연속 당김, 발사 시 추적탄 재장전.
- **교차 사격 리듬 (CROSSFIRE_RHYTHM)**: 방아쇠 1회(+조건부 추가).
- **퀵 랙 (QUICK_RACK)**: 추적탄 장전 + 드로우.
- **핫 챔버 (HOT_CHAMBER)**: 방아쇠 1회, 발사 성공 시 추가 피해.
- **체인 버스트 (CHAIN_BURST)**: 방아쇠 연속 당김, 첫 발 성공 시 추가 1회.
- **오버클록 드럼 (OVERCLOCK_DRUM)**: 지속. 매 턴 첫 추적탄 발사 시 추가 방아쇠 1회.
- **에칭 트레이서 (ETCHED_TRACER)**: 추적탄 장전 + 조건부 봉인 레벨 상승.
- **추적탄 변환 (TRACER_CONVERSION)**: 장전된 일반탄 최대 2발을 추적탄으로 변환(업그레이드 시 1드로우).

## 유틸/기타

- **엄폐 (TAKE_COVER)**: 방어도 + 빈 칸 수 보너스, 이번 턴 재장전 불가.
- **탄창 비우기 (EMPTY_THE_MAGAZINE)**: 실린더 전부 제거 + 탄종별 보너스(드로우/에너지/취약).
- **실린더 교체(근접) (CHAMBER_SWAP_NEXT)**: 현재 칸과 다음 칸 탄환 교체.
- **시질 가드 (SIGIL_GUARD)**: 기본 방어도 획득 + 봉인 레벨당 방어도 추가.
